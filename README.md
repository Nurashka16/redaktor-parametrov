# Редактор параметров

Тестовое задание на реализацию редактора параметров на React + TypeScript.

## Описание

Компонент `ParamEditor` позволяет редактировать структуру `Model` на основе списка параметров `params: Param[]` и начального состояния `model: Model`.

### Возможности:
- Отображение всех параметров из списка
- Инициализация значений из модели
- Редактирование значений через текстовые поля
- Получение актуальной модели через метод `getModel()`
- Поддержка только текстовых параметров (`type: "string"`)

### Технологии:
- React (классовые компоненты)
- TypeScript
- Jest + React Testing Library

## Установка и запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск тестов
```bash
npm test
```

### 3. Запуск в режиме разработки
```bash
npm run dev
```
